# fromflux/sierpinski-explorer
Sierpinski Triangle Explorer

This is the source code for the app available [here](https://sierpinski-explorer.herokuapp.com/).

## Installation

```
yarn install
```

## Development Server
```
# start local development server
yarn dev

# open in your browser
http://localhost:8080/
```
